\# Tavern Coffee House Landing Page



A coffee shop landing page built using HTML and CSS as part of my web development learning journey.



This project helped me practice creating real-world website layouts, working with Flexbox, styling components, and adding basic animations.



\## Features



\* Sticky navigation bar

\* Hero section with call-to-action buttons

\* Coffee product showcase

\* Featured content sections

\* Blog/news section

\* Footer with business information

\* Scroll animations using AOS



\## Technologies Used



\* HTML5

\* CSS3

\* Flexbox

\* Google Fonts

\* AOS (Animate On Scroll)
